Handshake Connect 2.0 CSS Framework

This is a set of html pages that will demonstrate the HS Connect 2.0 CSS Framework

HS Connect 2.0 is built from SASS Files provided by Bootstrap 4 and Kendo. These pages show how various Kendo Widgets and Bootstrap patterns will be rendered in a Handshake Portal

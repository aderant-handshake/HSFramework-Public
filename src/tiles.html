Action Tiles

Action Tiles display each action item as a separate tile. Tiles are expected to have a fixed width which will default to 320px, defined as $action-tile-width in the SASS build. This may be altered ad-hoc. The list below is contained in and element with class hs-action-tiles-list which will stack the action tiles.

The action center supports a smaller footprint of multiple items. The items will be arrange in a grid, with each item being 200px wide (set by the $action-center-item-width sass variable)

You can leave the border off the title by applying the class border-0; which works in most cases where there is a default border you don't want.

Action Group

hs-actions-tile-group will distribute the tiles horizontally. Each tile will be the height of the tallest item. The "todo" button will be at the bottom. These tiles use a class name "hs-action-sugar" which produces a folded corner effect when you hover over the tile.

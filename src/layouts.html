hs-grid-magazine defines a layout with the following elements:
  • sidebars (2)
  • optional header
  • optional footer
  • main content area

Additionally, you may use helper classes to alter the layout.

  • hs-grid-topbars: position the sidebars next to each other (50/50) above the main content.
  • hs-grid-content-left: position the main content to the left of the 2 sidebars rather than the right.
  • hs-grid-nogap : may be used on any hs-grid* container to make the grid gap zero at all viewport sizes
Of note, classes applied to the header element are bg-success, h1, m-0, p-2 and text-center

It will contain however many columns are in the markup, all will have an equal width. Though you may apply the classes hs-grid-double, triple or even quad to increase the width of an element

hs-tabstrip-full

Tabs will use the entire width of the parent container. Borders are removed, except for the bottom border of the active tab.

hs-tabstrip-uppercase

Sets the text of the tabs to be in Uppercase.

hs-tabstrip-bicolor

Sets the Active Tab to a background of $tabcolor(a), and all inactive tabs to $tabcolor(b). Text color is set using a function to determine optimim color based on the background-color. A slight box-shadow is set on the active tab.

hs-tabstrip-fullcolor

Sets the background color of each tab to $tabcolor(a..e), then repeats. The opacticy of inactive tabs is $tabcolor(opacity), while the opacity for active and onhover tabs is 1 A slight box-shadow is set on the active tab.

hs-tabstrip-rounded

Set a radius to the top left/right corners. Radius is defined in the $tabcolor(radius)

hs-tabstrip-radio

Renders the tabs as a set of radio control buttons

hs-tabstrip-shaded

Tabs will use full width of the parent container. Active tab will use $tabcolor(a) as the background. Hover backgroud will be $tabcolor(b).

hs-tabstrip-flat

Removes any box-shadow that may be set by another class such as hs-tabstrip-fullcolor.

hs-tabstrip-flush

Removes padding and border of child elements such as a HTML5Grid or a list-group inside an HTML5ListView when it is a direct deciendent to the active tab container

hs-tabstrip-compact

Renders a compact tab-strip with a simple underline of the active/hover item. Supports all theme colors for the underline hs-tabstrip-primary (default), hs-tabstrip-secondary, hs-tabstrip-success, etc...

TIPS
  • Classes may be combined like fullwidth & fullcolor, but not all will make sense like fullcolor and bicolor.
  • For consistency in your site, like all tabs should be fullwidth and upper case, use an Application Option to set the desired classes.
    Application Option: tabclasses = "hs-tabstrip-fullwidth hs-tabstrip-uppercase"
